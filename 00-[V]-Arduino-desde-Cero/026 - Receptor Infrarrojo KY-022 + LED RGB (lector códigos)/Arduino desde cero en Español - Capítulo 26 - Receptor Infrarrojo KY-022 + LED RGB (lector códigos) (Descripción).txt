Analizaremos la comunicación por infrarrojo con el módulo KY-022 para comandar un LED RGB, todo conectado a Arduino, mediante un control remoto convencional.
Se explicará el concepto de la comunicación mediante luz infrarroja y como obtener el código asociado a cada botón (decodificado) del control remoto para tomar distintas acciones.
