Mediante la Wemos D1 Mini basada en ESP8266 iniciaremos un servidor web y utilizaremos información del encabezado para encender un LED conectado a un pin digital de Arduino y también crearemos una página web para mostrar temperatura y humedad de un sensor DHT11.
Continuando con lo iniciado en el capitulo 41, automatizaremos la conexión a la red local, con el método GET daremos el comando para enceder o apagar un LED y obtener los datos de un sensor DHT11 y mostrarlos en el navegador web.
