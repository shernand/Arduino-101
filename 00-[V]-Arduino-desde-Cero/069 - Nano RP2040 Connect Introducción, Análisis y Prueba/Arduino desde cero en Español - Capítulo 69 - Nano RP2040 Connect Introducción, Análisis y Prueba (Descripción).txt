La moderna placa de Arduino Nano RP2040 Connect con corazón de Raspberry Pi, conectividad WiFi, Bluetooth y gran cantidad de sensores incorporados hace de una placa de desarrollo versátil y fácil de utilizar en infinidad de proyectos.
Haremos un análisis detallado de su hardware, instalación de software para trabajar con el IDE de Arduino y un programa de demostración para el control del LED RGB incorporado mediante WiFi a través del navegador web de un teléfono móvil. Sentando las bases para futuros capítulos donde se analizarán puntualmente el micrófono PDM, el IMU con Machine Learning, Bluetooth y más.
