Con el módulo TCS3200 podremos determinar el color de una superficie de manera simple con Arduino, incluye iluminación incorporada y un sensor de 64 fotodiodos para la tarea.
Mediante una simple programación veremos como detectar los colores rojo, verde y azul en la superficie de papeles para tomar una acción, útil para procesos de automatización.


Código fuente de los programas visto en:
https://github.com/bitwiseAr/Curso-Arduino-desde-cero/
