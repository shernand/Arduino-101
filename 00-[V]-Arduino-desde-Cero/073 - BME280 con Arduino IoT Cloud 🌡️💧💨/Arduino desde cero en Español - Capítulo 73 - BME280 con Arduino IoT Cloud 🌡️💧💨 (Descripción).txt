En este proyecto de Internet de las Cosas integramos un sensor BME280 para la medición de temperatura, humedad y presión, conectamos con el servicio gratuito de Arduino IoT Cloud en la nube y mostramos la información de forma gráfica con medidores y tabla de historial.
De manera muy simple con una placa Nano RP2040 Connect utilizamos un sensor BME280 mediante bus I2C y configuramos todo en Arduino IoT Cloud, con una mínima cantidad de código logramos obtener las mediciones y crear un Dashboard o Panel de Control con medidores y una tabla con el historial de temperatura.

También vemos como ingresar mediante teléfono móvil con la aplicación Arduino IoT Cloud Remote.
