El futuro del IDE (Entorno de Desarrollo Integrado) de Arduino ya está aquí cin su versión 2, lo instalamos desde cero y analizamos todas las nuevas características visuales y de productividad del nuevo editor con diferentes placas (Uno y Nano 33 BLE).
Veremos que puede coexistir sin problemas con la versión anterior 1.8.19 y las novedades del editor a la hora de sugerir funciones, acceso rápido a referencias, autocompletado y mucho más en un entorno moderno y personalizable.
Basado en el framework (marco de trabajo) Eclipse Theia es un proyecto de código abierto para desarrollo de IDEs de escritorio y en la nube, con una experiencia de uso similar al VS Code (Visual Studio Code).
