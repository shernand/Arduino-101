Conectaremos Arduino por WiFi mediante la Wemos D1 Mini basada en ESP8266 y mediante comandos AT podremos establecer un servidor web para responder peticiones desde un navegador web.

Veremos como el SoC ESP8266 permite conectar a Arduino fácilmente a una red local mediante WiFi, iniciar un servidor web para recibir información mediante el método GET y responder peticiones del navegador web, todo mediante simples comandos AT. 

Código fuente del programa visto en:
https://github.com/bitwiseAr/Curso-Arduino-desde-cero/
